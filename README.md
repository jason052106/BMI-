# BMI 計算機

## 📝 專案簡介
這是一個使用 C# Windows Forms 開發的 BMI 計算機應用程式。本專案除了具備基本的 BMI 數值計算與體位判定功能外，還特別強化了「視窗介面設計」與「防呆機制」，以提供更流暢的使用者體驗。

## ✨ 功能特色
1.輸入身高(cm)與體重(kg)即可計算 BMI，並依據標準顯示對應的體位狀態（過輕、健康、肥胖等）。

2.根據不同的 BMI 體位，自動切換自訂的柔和背景色號。

3.介面版面固定，禁止隨意拉伸，確保排版整齊。

4.限制身高與體重欄位僅能輸入「數字」與「小數點」，防止使用者輸入錯誤字元導致程式崩潰。

5.提供「清除」按鈕，方便使用者快速清空數據重新輸入。
